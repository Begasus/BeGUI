    The Preferences Class controls the storage and retrieval of all application specific preference data. When you build an application class (f.i. class MyApp, see MyApp.h) you base the class publicly first on BApplication, then on The Preferences Class .

    The Preference Panel has gadgets allowing access to all the preference settings. From here you can access The Font Preference Panel with it's The Font Display Panel, and the various Gadget Specific Panels.

    The Font Display Panel is accessed via the The Preference Paneland simply displays the alphabet and digits so you may see what your choice of font looks like.

    The Gadget Specific Panels allow you to adjust various aspects of the gadgets, like their colors. Each panel is specific to a certain gadget type, and may have additional controls besides color. Each panel has a demonstration item so that you may view the results of your actions in real time.
