    Overview

    [method summary]

    The MyPopUpMenuData Class provides data for the MyPopUpMenu class.

    MyPopUpMenuData()

      MyPopUpMenuData(const char * paramLabel,
        BList * paramList,
        bool paramSetMark = false);

      This is the constructor. The first parameter paramLabel, is menu item string.

      The parameter paramList is the list of string items which will be attached to the BPopUpMenu which contains this item. The parameter paramSetMark is used to mark an item when displayed.

      ~MyPopUpMenuData();

      This is the destructor. It does nothing.
