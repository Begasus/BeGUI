    [method summary]

    The MyRadioViewData Class provides data for MyRadioView.

    MyRadioViewData()

      MyRadioViewData(uint32 paramMessageValue,
        const char * paramLabel,
        BList * paramList,
        bool paramSelected = false);

      This is the constructor. The first parameter paramMessageValue, is used to create the message this button sends.

      The parameter paramLabel is the text string label of the button.

      The parameter paramList contains a BList of BRadioButtons to which this will be added.

      The parameter paramSelected is set to true if this button is the one initially selected. If all buttons are set to false, the first one on the list will be selected by default.

      This is the destructor. It does nothing.
