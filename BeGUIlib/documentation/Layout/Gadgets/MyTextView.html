The MyTextView Class

    Derived from: BTextView

    Declared in: myTextView.h

    Library: libBeGUI.so

    Overview

    [method summary]

    The MyTextView Class is used as a simple (or not so simple) multi-line text entry box. It is a wrapper for the BeOS BTextView class.

    MyTextView()

      MyTextView(const char * paramName,
        const char * paramText,
        float paramMaxWidth,
        float paramMaxHeight,
        LayoutMatrix * paramLayoutMatrix);

      This is the constructor. The parameter paramName is passed on to the BTextView. The parameter paramText is the text with which the gadget initially opens (it may be NULL).

      The third and next parameters, paramMaxWidth and paramMaxHeight control the width and height of the text box.

      The parameter paramLayoutMatrix is the LayoutMatrix to which this gadget will be attached.

      There is a bit of code in this CTOR, it most all deals with text, filling the box and setting the style, size and color. The text entry region is limited to not being resizable, as doing so could cover up other gadgets.

    ~MyTextView()

      ~MyTextView();

      This is the destructor. It does nothing.

    AttachedToWindow()

      void AttachedToWindow(void);

      This just overrides the BTextControl method to provide color control, it calls the inherited version.

    MyGetPreferredSize()

      void MyGetPreferredSize(float * paramWidth,
        float * paramHeight)

      The BTextView class's method GetPreferredSize() has some problems, so I have provided a replacement which works for my needs.
