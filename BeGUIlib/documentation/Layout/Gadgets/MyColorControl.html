The MyColorControl Class

    Derived from: BColorControl

    Declared in: myColorControl.h

    Library: libBeGUI.so

    Overview

    [method summary]

    The MyColorControl Class is used as a color selection tool. It is a wrapper for the BeOS BColorControl class.

    MyColorControl()

      MyColorControl(color_control_layout paramColorControlLayout,
        const char * paramName,
        uint32 paramMsg,
        LayoutMatrix * paramLayoutMatrix,
        float paramCellSize = 1,
        bool paramUseOffscreen = true);

      This is the constructor. Almost everything is passed directly in to the base class. The parameter paramLayoutMatrix is the LayoutMatrix to which this gadget will be attached.

    ~MyColorControl()

      ~MyColorControl();

      This is the destructor. It does nothing.

    AttachedToWindow()

      void AttachedToWindow(void);

      This just overrides the BColorControl method to provide color control, it calls the inherited version.
