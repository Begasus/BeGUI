The MySlider Class

    Derived from: BSlider

    Declared in: mySlider.h

    Library: libBeGUI.so

    Overview

    [method summary]

    The MySlider Class is used as a wrapper for the BSlider class.

    MySlider()

      MySlider(BRect paramFrame,
        const char * paramName,
        const char * paramLabel,
        uint32 paramMessageValue,
        int32 paramMinValue,
        int32 paramMaxValue,
        float paramPosition,
        LayoutMatrix * paramLayoutMatrix);

      This is the constructor. This sets the initial position of the knob from paramPosition, otherwise all parameters are just passed along to the BSlider base class.

    ~MySlider()

      ~MySlider();

      This is the destructor. It does nothing.

    AttachedToWindow()

      void AttachedToWindow(void);

    This sets the preferred thumb (knob) style, sets the preferred font, and sets the preferred colors.
