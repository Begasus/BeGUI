The MyCheckBox Class

    Derived from: BCheckBox

    Declared in: myCheckBox.h

    Library: libBeGUI.so

    Overview

    [method summary]

    The MyCheckBox Class is used as a check box. It is a wrapper for the BeOS BCheckBox class.

    MyCheckBox()

      MyCheckBox(const char * paramName,
        const char * paramLabel,
        int32 i32Value,
        uint32 uparamMsg,
        LayoutMatrix * paramLayoutMatrix);

      This is the constructor. The first, second and next parameters, paramName, paramLabel and uparamMsg are passed on to the BCheckBox class.

      The parameter paramLayoutMatrix is the LayoutMatrix to which this gadget will be attached.

      There is a tiny bit of code in this CTOR, it sets the value and sets the font to the applications preferred font.

    ~MyCheckBox()

      ~MyCheckBox();

      This is the destructor. It does nothing.

    AttachedToWindow()

      void AttachedToWindow(void);

      This just overrides the BCheckBox method to provide color control, it calls the inherited version.
