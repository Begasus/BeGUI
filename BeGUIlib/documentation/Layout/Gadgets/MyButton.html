The MyButton Class

    Derived from: BButton

    Declared in: myButton.h

    Library: libBeGUI.so

    Overview

    [method summary]

    The MyButton Class is used as a simple button gadget. It is a wrapper for the BeOS BButton class.

    MyButton()

      MyButton(const char * paramName,
        const char * paramLabel,
        uint32 uparamMsg,
        LayoutMatrix * paramLayoutMatrix,
        bool paramSetEnabled = true,
        int paramOnOff = B_CONTROL_OFF);

      This is the constructor. The first three parameters, paramName, paramLabel and uparamMsg are passed on to the BButton class.

      The parameter paramLayoutMatrix is the LayoutMatrix to which this gadget will be attached.

      The parameter paramSetEnabled causes the button to be enabled initially if true, as it is by default. The parameter paramOnOff controls whether the button is displayed in it's "off" state (ie. unselected, as it is by default) when first displayed.

      There is a tiny bit of code in this CTOR, it sets the font, and the on/off and enabled states of the button.

    ~MyButton()

      ~MyButton();

      This is the destructor. It does nothing.

    MyGetPreferredSize()

      void MyGetPreferredSize(float * paramWidth,
        float * paramHeight)

      The BButton class's method GetPreferredSize() has some problems, so I have provided a replacement which works for my needs.

    AttachedToWindow()

      void AttachedToWindow(void);

      This just overrides the BButton method to provide color control, it calls the inherited version.
