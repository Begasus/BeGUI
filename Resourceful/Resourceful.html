Resourceful

This is a command line only application. When run it simply prints the contents of one of it's resourse strings to the Terminal. This is useful soley as a test of resource editors like ResEDgar. Run it once to see what resource string it prints; edit it's resources, change that string; run it again and see that your choice was respected!
